# 核心引擎

## 职责

PetCore 的核心引擎（`petcore/internal/core/`），负责状态机运行、事件循环和内置动作管理。是整个 PetCore 的 "main loop"。

## 核心类/接口

| 文件路径 | 类/接口 | 职责 |
|---------|---------|------|
| `petcore/internal/core/engine.go` | `Engine` | 主引擎，事件循环 + sidecar 通信 |
| `petcore/internal/core/state.go` | `State` | 状态定义 + 转换规则 |
| `petcore/internal/core/actions.go` | `Action` | 内置动作（待机动作、情绪表达等） |
| `petcore/internal/event/event.go` | `Event` / `EventType` | 事件类型定义 |
| `petcore/internal/event/sink.go` | `Sink` | 事件消费接口 |

## 事件类型

```go
const (
    EventStateChanged  EventType = "state.changed"
    EventPetSpeak      EventType = "pet.speak"
    EventPetAction     EventType = "pet.action"
    EventPetEmotion    EventType = "pet.emotion"
    EventAgentThinking EventType = "agent.thinking"
    EventAgentReply    EventType = "agent.reply"
    EventMemoryUpdate  EventType = "memory.updated"
    EventError         EventType = "error"
)
```
