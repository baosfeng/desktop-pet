# 核心引擎

## 职责

PetCore 的核心引擎（`petcore/internal/core/`），负责状态机运行、事件循环和内置动作管理。是整个 PetCore 的 "main loop"。

## 核心类/接口

| 文件路径 | 类/接口 | 职责 |
|---------|---------|------|
| `petcore/internal/core/engine.go` | `Engine` | 主引擎，DI 组装所有模块，Run/HandleInput/GetStatus |
| `petcore/internal/event/event.go` | `Event` / `Type` | 事件类型定义 |
| `petcore/internal/event/event.go` | `Sink` / `NoopSink` | 事件消费接口 |

## 事件类型

```go
const (
    EventStateChanged  Type = "state.changed"
    EventPetSpeak      Type = "pet.speak"
    EventPetAction     Type = "pet.action"
    EventPetEmotion    Type = "pet.emotion"
    EventAgentThinking Type = "agent.thinking"
    EventAgentReply    Type = "agent.reply"
    EventMemoryUpdated Type = "memory.updated"
    EventError         Type = "error"
)
```
