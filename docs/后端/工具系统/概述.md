# 工具系统

## 职责

提供 AI Agent 可调用的工具集合。工具通过统一注册中心管理，支持 MCP 协议扩展和内置工具。Agent 在对话循环中动态调用工具并注入结果。

## 核心类/接口

| 文件路径 | 类/接口 | 职责 |
|---------|---------|------|
| `petcore/internal/tool/registry.go` | `Registry` | 工具注册中心（注册/查找/执行） |
| `petcore/internal/plugin/mcp_client.go` | `MCPClient` | MCP 协议客户端（L3 插件） |
| `petcore/internal/tool/builtin/web_search.go` | `WebSearch` | 联网搜索工具 |
| `petcore/internal/tool/builtin/desktop_ctrl.go` | `DesktopCtrl` | 桌面控制工具（Phase 6） |

## 工具注册

```go
// 内置工具在 init() 中注册
func init() {
    Register(Tool{
        Name:        "web_search",
        Description: "搜索互联网信息",
        Parameters:  schema{...},
        Execute:     func(ctx Context, args Args) (string, error) {
            // 调用搜索 API
        },
    })
}

// MCP 工具由 MCPClient 在运行时自动发现
mcpClient := NewMCPClient(cmd)
tools, err := mcpClient.ListTools()
```

## 工具调用循环

Agent 在执行对话时自动管理工具调用：

```
LLM 回复 → 解析 Tool Call
               │
               ▼
        Registry.Execute(name, args)
               │
               ▼
        执行内置工具 / MCP 工具
               │
               ▼
        结果注入 → 继续 LLM 调用
               │
               ▼
        (重复直到无工具调用)
```

## 业务场景映射

| 需求关键词 | 入口 | 说明 |
|-----------|------|------|
| 搜索、联网 | `WebSearch` | 内置 web_search 工具 |
| 桌面控制 | `DesktopCtrl` | 桌面控制工具（Phase 6） |
| MCP、工具扩展 | `MCPClient` | 外部 MCP 工具服务器 |
| 工具注册 | `Registry.Register()` | 注册新工具 |
