# 构建与测试

## 测试执行要求

> **提交前必须运行全部测试，修复所有失败的测试用例，确保全部通过。**

- 每次提交前都应当运行项目对应的测试命令
- 如果存在测试失败，必须修复后才能提交

## 测试命令

### 前端测试
```bash
cd frontend
npm test
```

### 后端测试
```bash
cd backend
python -m pytest
```

## 构建命令

### 前端构建
```bash
cd frontend
npm run build
```

### Python 后端编译
```bash
cd backend
pyinstaller --onefile --name backend-bin app/server.py
```

### 完整打包
```bash
cd frontend
npm run build
npx electron-builder
```
