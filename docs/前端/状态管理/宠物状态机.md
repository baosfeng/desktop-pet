# 宠物状态机

## 职责

管理宠物的行为状态流转：待机→关注→交互→思考。状态机驱动 Live2D 的动画选择和 UI 展示。

## 状态定义

```
                        ┌──────────┐
          ┌────────────►│   待机    │◄──────────────┐
          │             │ (随机动作) │               │
          │             └─────┬────┘               │
          │                   │                    │
      对话结束              检测到用户             超时
          │                   │                    │
          │             ┌─────▼────┐               │
          ├────────────►│   关注    │───────────────┤
          │             │ (转头/看) │               │
          │             └─────┬────┘               │
          │                   │                    │
          │              用户对话/点击               │
          │                   │                    │
          │             ┌─────▼────┐               │
          │             │   交互    │────思考中──────┤
          │             │ (对话/玩) │  (等待回复)    │
          │             └──────────┘               │
          └────────────────────────────────────────┘
```

## 核心 Store

| 文件路径 | Store | 职责 |
|---------|-------|------|
| `frontend/src/store/petState.ts` | `usePetState` | 宠物状态管理 |

## 状态转换逻辑

| 当前状态 | 触发事件 | 下一状态 | 行为 |
|---------|---------|---------|------|
| 待机 | 鼠标悬停/光标靠近 | 关注 | 播放「发现用户」动画 |
| 关注 | 用户对话/点击 | 交互 | 播放「准备对话」动画 |
| 交互 | 发送消息 | 思考 | 播放「思考中」动画 |
| 思考 | 收到回复 | 交互 | 播放「回复」动画 |
| 交互 | 超时（30s 无操作） | 待机 | 随机待机动作 |
