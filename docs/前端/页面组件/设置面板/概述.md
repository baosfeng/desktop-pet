# 设置面板

## 职责

提供宠物配置的图形化界面，包括 LLM 配置、角色人设和窗口透明度设置。设置通过 `localStorage` 持久化。

## 核心组件

| 文件路径 | 组件 | 职责 |
|---------|------|------|
| `frontend/src/components/SettingsPanel.tsx` | `SettingsPanel` | 设置面板主组件 |

## Props 接口

```typescript
interface SettingsPanelProps {
  onClose: () => void;
}
```

## 设置项

| 类别 | 设置 | 说明 |
|------|------|------|
| LLM 配置 | API Key | OpenAI 兼容 API 的密钥（password 输入框） |
| | Base URL | API 地址（默认 https://api.openai.com/v1） |
| | 模型名 | 使用的模型（默认 gpt-4o-mini） |
| 角色人设 | 性格描述 | 宠物的性格设定（textarea） |
| 通用 | 透明度 | 窗口透明度滑块（10%-100%） |

## 存储

- 键：`desktop-pet-settings`（localStorage）
- 默认值在 `DEFAULT_SETTINGS` 中定义
- 加载失败时回退到默认值（不阻塞启动）
