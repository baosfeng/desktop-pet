# 设置面板

## 职责

提供宠物配置的图形化界面，包括 LLM 配置、角色人设、记忆管理、模型选择和通用设置。

## 配置项 {待补充}

## 核心组件

| 文件路径 | 组件 | 职责 |
|---------|------|------|
| `frontend/src/components/SettingsPanel.tsx` | `SettingsPanel` | 设置面板主组件 |

## 设置项

| 类别 | 设置 | 说明 |
|------|------|------|
| LLM 配置 | API Key | OpenAI 兼容 API 的密钥 |
| | Base URL | API 地址 |
| | 模型名 | 使用的模型（如 gpt-4o-mini） |
| 角色人设 | 性格描述 | 宠物的性格设定 |
| | 说话风格 | 说话的语气和用词 |
| | 角色名称 | 宠物的名字 |
| 记忆管理 | 查看记忆 | 浏览存储的记忆 |
| | 编辑记忆 | 修改/删除记忆 |
| | 清除记忆 | 清除所有记忆 |
| 模型选择 | 当前模型 | 当前 Live2D 模型 |
| | 下载模型 | 在线下载更多模型 |
| | 导入模型 | 导入本地模型 |
| 通用 | 开机自启 | 登录时自动启动 |
| | 透明度 | 宠物透明度滑块 |
| | 音量 | 语音输出音量 |
