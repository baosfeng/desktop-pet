# 窗口管理器

## 职责

管理与 Tauri 窗口相关的创建、配置、鼠标穿透、位置记忆和拖拽交互。确保宠物窗口始终在正确位置，不影响用户正常操作。

> 窗口管理由 Tauri Rust 壳（`src-tauri/src/window.rs`）负责，前端通过 Tauri IPC（`bridge.ts`）控制。

## 核心功能

- **透明无边框窗口**：窗口背景完全透明，仅显示 Live2D 和 UI（Tauri 配置 `transparent: true, decorations: false`）
- **鼠标穿透**：默认状态下鼠标点击穿透到下层应用（Tauri `set_ignore_cursor_events(true)`）
- **位置记忆**：关闭时保存窗口位置到 `window_position.json`，启动时恢复
- **多屏边界保护**：窗口位置超出屏幕时自动限制到可见区域（x ≥ 0, y ≥ 0）
- **置顶控制**：`alwaysOnTop: true` 始终置顶

## 核心类/接口

| 文件路径 | 类/函数 | 职责 |
|---------|---------|------|
| `src-tauri/src/window.rs` | `create_pet_window()` | 窗口创建与位置恢复 |
| `src-tauri/src/window.rs` | `toggle_clickthrough()` | 切换鼠标穿透状态 |
| `src-tauri/src/window.rs` | `get_window_position()` | 获取窗口当前位置 |
| `src-tauri/src/window.rs` | `set_window_position()` | 设置窗口位置并持久化 |
| `src-tauri/src/commands.rs` | `5 Tauri 命令` | 前端通过 invoke 调用（含 chat/get_status） |
| `frontend/src/lib/bridge.ts` | `Bridge` | Tauri IPC 封装 |

## Tauri 命令

| 命令名 | 说明 |
|--------|------|
| `toggle_clickthrough` | 切换鼠标穿透（true=穿透/待机，false=可交互） |
| `get_window_position` | 获取窗口位置 |
| `set_window_position` | 设置窗口位置 |
| `chat` | 发送对话消息到 PetCore |
| `get_status` | 获取宠物状态 |

## 鼠标穿透状态

```
待机状态 → ignoreCursorEvents = true（穿透）
              │
              ▼
    鼠标进入窗口区域
              │
              ▼
    ignoreCursorEvents = false（可交互）
              │
       ┌──────┴──────┐
       ▼              ▼
    点击/拖拽        对话触发
       │              │
       ▼              ▼
    进入交互状态    进入对话状态
       │              │
       └──────┬──────┘
              ▼
    超时 (5秒无操作)
              │
              ▼
    恢复穿透 → 回到待机
```

```json
  }

```

## 边界情况

| 场景 | 期望行为 |
|------|---------|
| 窗口位置为负数 | 限制在屏幕左上角 (0,0) |
| 窗口超出屏幕右/下边缘 | 回退到屏幕可见区域 |
| 快速连续调用 toggle_clickthrough | 不 panic，状态正确切换 |
| 窗口未创建时调用 | 返回明确的错误 |
