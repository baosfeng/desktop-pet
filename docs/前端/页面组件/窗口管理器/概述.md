# 窗口管理器

## 职责

管理与 Tauri 窗口相关的创建、销毁、位置保存、透明模式、鼠标穿透和拖拽交互。确保宠物窗口始终在正确位置，不影响用户正常操作。

> 窗口管理由 Tauri Rust 壳（`src-tauri/src/window.rs`）负责，前端通过 Tauri IPC 控制。

## 核心功能

- **透明无边框窗口**：窗口背景完全透明，仅显示 Live2D 和 UI（Tauri 配置 `transparent: true, decorations: false`）
- **鼠标穿透**：默认状态下鼠标点击穿透到下层应用（Tauri `ignoreCursorEvents: true`）
- **拖拽模式**：用户拖拽宠物时临时关闭穿透，放下后恢复（Tauri IPC 动态切换）
- **位置记忆**：记住窗口位置，下次启动恢复
- **多屏支持**：在多显示器间跟随宠物位置
- **置顶控制**：`alwaysOnTop: true` 始终置顶

## 核心类/接口

| 文件路径 | 类/接口 | 职责 |
|---------|---------|------|
| `src-tauri/src/window.rs` | `WindowManager` | 窗口创建、透明、穿透、置顶、位置管理 |
| `src-tauri/src/tray.rs` | `TrayManager` | 系统托盘图标、右键菜单 |
| `src-tauri/src/commands.rs` | `Commands` | Tauri 命令（前端通过 invoke 调用） |
| `frontend/src/lib/bridge.ts` | `Bridge` | Tauri IPC 封装（替代旧 WebSocket） |
| `frontend/src/hooks/usePetEvents.ts` | `usePetEvents` | 监听 Tauri 事件（状态变化/动作/情绪） |

## 交互事件

| 事件 | 触发 | 行为 |
|------|------|------|
| 鼠标悬停 | 光标移到宠物上 | 关闭穿透（Tauri 动态设置），宠物看向光标 |
| 鼠标拖拽 | 按住拖拽 | 物理反馈（JS 拖拽计算），调用窗口移动命令 |
| 鼠标离开 | 光标移出窗口 | 恢复穿透 |
| 托盘点击 | 系统托盘图标 | 唤出/隐藏宠物窗口 |

## 鼠标穿透状态切换

```
待机状态 → ignoreCursorEvents = true
              │
              ▼
    鼠标进入窗口区域
              │
              ▼
    ignoreCursorEvents = false → 宠物产生反应
              │
       ┌──────┴──────┐
       ▼              ▼
    点击/拖拽        对话触发
       │              │
       ▼              ▼
    进入交互状态    进入对话状态
       │              │
       └──────┬──────┘
              ▼
    超时 (5秒无操作)
              │
              ▼
    恢复穿透 → 回到待机
```

```json
      "transparent": true,
      "decorations": false,
      "alwaysOnTop": true,
      "focus": false
    }]
  }
}
```
