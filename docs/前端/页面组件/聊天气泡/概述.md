# 聊天气泡

## 职责

显示 AI 回复的浮动聊天气泡 UI，跟随宠物位置，支持打字机效果和 Markdown 渲染。

## 核心功能

- **浮动显示**：气泡跟随宠物窗口位置，不遮挡宠物
- **打字机效果**：文本逐字显示，模拟说话节奏
- **Markdown 渲染**：支持加粗、列表、链接等格式
- **情感标签**：显示当前宠物情感状态的表情符号
- **历史回看**：滚动查看最近对话记录
- **自动消失**：一定时间后自动隐藏

## 核心组件

| 文件路径 | 组件 | 职责 |
|---------|------|------|
| `frontend/src/components/SpeechBubble.tsx` | `SpeechBubble` | 聊天气泡主组件 |

## 消息流

```
后端 → WebSocket: { type: "pet.speak", data: { text: "...", emotion: "happy" } }
                                          │
                                          ▼
SpeechBubble: 显示文本（打字机效果）+ 情感表情
```

## 交互

- 用户点击气泡可关闭
- 鼠标悬停保持显示
- 新消息自动滚动到底部
