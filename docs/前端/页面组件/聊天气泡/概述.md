# 聊天气泡

## 职责

显示 AI 回复的浮动聊天气泡 UI，支持打字机效果和 Markdown 渲染，通过 Tauri IPC 接收后端事件。

## 核心功能

- **浮动显示**：气泡跟随宠物窗口位置
- **打字机效果**：文本逐字显示，模拟说话节奏（`TYPEWRITER_SPEED_MS = 30ms`）
- **Markdown 渲染**：使用 `react-markdown` 支持加粗、列表、链接等格式
- **流式显示**：新消息自动滚动到底部
- **输入框**：支持 Enter 发送和 Shift+Enter 换行

## 核心组件

| 文件路径 | 组件/函数 | 职责 |
|---------|----------|------|
| `frontend/src/components/ChatBubble.tsx` | `ChatBubble` | 聊天气泡主组件，消息列表 + 输入框 |

## 消息流

```
PetCore (sidecar stdout) → Rust 壳解析 → app.emit("pet:event")
                                          │
                                          ▼
bridge.ts onPetEvent → App.tsx setMessages
                                          │
                                          ▼
ChatBubble 显示文本（打字机效果）+ Markdown 渲染
```

## Props 接口

```typescript
interface ChatBubbleProps {
  messages: Message[];      // 消息列表
  onSendMessage: (text: string) => void;  // 发送消息回调
}

interface Message {
  id: string;
  role: "user" | "assistant" | "system";
  content: string;
  timestamp: number;
}
```

## 交互

- 用户按 Enter 发送消息
- 新消息自动滚动到底部
- 最新 AI 回复显示打字机光标 `|`
