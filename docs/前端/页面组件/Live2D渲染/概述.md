# Live2D 渲染

## 职责

加载和渲染 Live2D 模型（Cubism 4），管理动画播放、表情切换、口型同步和触碰反应。

## 核心功能

- **模型加载**：支持 Cubism 4 的 `.model3.json` 格式
- **动画播放**：待机动画、动作动画、表情动画
- **表情切换**：根据 AI 回复的情感标签切换表情（高兴/难过/生气/平静/兴奋）
- **口型同步**：TTS 语音输出时驱动嘴部动画
- **触碰反应**：点击/拖拽时触发特定动作
- **多模型管理**：支持按角色切换不同模型

## 核心类/接口

| 文件路径 | 类/组件 | 职责 |
|---------|---------|------|
| `frontend/src/components/Live2DView.tsx` | `Live2DView` | 主渲染组件，PixiJS 画布管理 |
| `frontend/src/models/ModelManager.ts` | `ModelManager` | 模型加载/切换/缓存 |

## 动画控制

```
AI 回复 "我今天很开心！" → JSON { emotion: "happy" }
                                 │
                                 ▼
Live2DView → ModelManager.playAnimation("happy")
                                 │
                                 ▼
                Live2D 模型播放开心表情 + 动作
```

## 事件

| 消息类型 | 方向 | 说明 |
|---------|------|------|
| `pet.action` | 后端→前端 | 播放指定动画 |
| `pet.emotion` | 后端→前端 | 切换表情状态 |
| `pet.thinking` | 后端→前端 | 显示/隐藏思考动画 |
