# Live2D 渲染

## 职责

加载和渲染 Live2D 模型（Cubism 4），管理动画播放、表情切换和触碰反应。

> **当前状态：** 已集成 pixi-live2d-display + PixiJS 8，支持 CDN 和本地模型加载。

## 核心功能

- **模型加载**：支持 Cubism 4 的 `.model3.json` 格式（待实现）
- **动画播放**：待机动画、动作动画、表情动画（待实现）
- **表情切换**：根据 AI 回复的情感标签切换表情（待实现）
- **触碰反应**：点击/拖拽时触发特定动作（待实现）
