# 构建与打包

## 开发构建

```bash
cd frontend
npm install
npm run dev    # 开发模式
```

## 生产打包

```bash
# 编译前端
npm run build

# 打包为 macOS .app
npx electron-builder

# 产物在 dist/ 目录
```

## 打包配置

`electron-builder` 配置位于 `frontend/electron-builder.yml`，关键配置：

```yaml
appId: com.desktop-pet.app
mac:
  target: dmg
  icon: assets/icon.icns
extraResources:
  - from: ../backend/dist/backend-bin
    to: backend-bin
```

## 自动更新

```bash
# 发布新版本（需要 GitHub Token）
npm run release
```

- 使用 `electron-updater` 实现增量更新
- 自动从 GitHub Releases 下载差异包
- 支持版本回滚

## 体积预算

| 组件 | 大小 |
|------|------|
| Electron 壳 | ~150-180 MB |
| PyInstaller 编译产物 | ~50-80 MB |
| 前端代码 | ~2-5 MB |
| Live2D 基础模型 | ~3-5 MB |
| **总计** | **~210-260 MB** |
