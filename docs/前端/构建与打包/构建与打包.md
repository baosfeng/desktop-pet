# 构建与打包

> 技术栈：Tauri v2 (Rust) + React 18 + PixiJS 8 + Go PetCore

## 开发构建

```bash
# 终端 1：Go PetCore (独立运行调试)
cd petcore
go run ./cmd/petcore/ --mode sidecar

# 终端 2：前端热更新
cd frontend
pnpm install
pnpm dev

# 一键启动（构建前端 + 编译 Go + 运行 Rust 壳）
cd src-tauri
cargo tauri dev
```

## 生产打包

```bash
# 1. 编译 Go PetCore (CGO_ENABLED=0，跨平台编译)
cd petcore && \
CGO_ENABLED=0 GOOS=darwin GOARCH=arm64 \
  go build -ldflags="-s -w" \
  -o ../src-tauri/binaries/petcore-aarch64-apple-darwin ./cmd/petcore/

# 2. 构建前端
cd frontend
pnpm build

# 3. 打包为 macOS .app（自动包含 petcore + 前端）
cd ../src-tauri
cargo tauri build

# 产物：src-tauri/target/release/bundle/dmg/DesktopPet-1.0.0.dmg (~25MB)
```

## 打包配置

Tauri 配置位于 `src-tauri/tauri.conf.json`，关键配置：

```json
{
  "app": {
    "windows": [{
      "label": "pet",
      "transparent": true,
      "decorations": false,
      "alwaysOnTop": true,
      "focus": false,
      "width": 400,
      "height": 600
    }]
  },
  "bundle": {
    "externalBin": ["binaries/petcore"],
    "icon": ["icons/icon.icns"]
  }
}
```

## 自动更新

使用 `tauri-plugin-updater`（Tauri 原生 HTTP API 更新器），非增量差异包：

```bash
# 发布新版本（需要 GitHub Token + 签名密钥）
cargo tauri build
# 将产物上传至 GitHub Releases 页面
# 自动更新配置在 tauri.conf.json 中声明
```

- 启动时 + 运行时每 6 小时检查更新
- 检测到更新后立即后台下载
- 用户确认后重启安装（不强制）
- 启动崩溃自动回滚

## 体积预算

| 组件 | 大小 |
|------|------|
| Tauri 壳（Rust + WebView） | ~10-15 MB |
| Go PetCore 静态二进制 | ~8-12 MB |
| 前端编译产物（React + PixiJS） | ~2-5 MB |
| Live2D 基础模型 | ~3-5 MB |
| **.app 首次下载** | **~23-37 MB** |

## 交叉编译

### macOS (Apple Silicon)

```bash
CGO_ENABLED=0 GOOS=darwin GOARCH=arm64 \
  go build -o ../src-tauri/binaries/petcore-aarch64-apple-darwin ./cmd/petcore/
```

### macOS (Intel)

```bash
CGO_ENABLED=0 GOOS=darwin GOARCH=amd64 \
  go build -o ../src-tauri/binaries/petcore-x86_64-apple-darwin ./cmd/petcore/
```

### Windows

```bash
CGO_ENABLED=0 GOOS=windows GOARCH=amd64 \
  go build -o ../src-tauri/binaries/petcore-x86_64-pc-windows-msvc.exe ./cmd/petcore/
```

### Linux

```bash
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 \
  go build -o ../src-tauri/binaries/petcore-x86_64-unknown-linux-gnu ./cmd/petcore/
```
