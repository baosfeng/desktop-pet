# 原生功能

## 桌面端原生能力

| 能力 | 实现方式 | 用途 |
|------|---------|------|
| 透明窗口 | `BrowserWindow` transparent 选项 | 宠物窗口无背景 |
| 鼠标穿透 | `win.setIgnoreMouseEvents()` | 点击穿透到底层 |
| 置顶 | `win.setAlwaysOnTop()` | 宠物始终可见 |
| 开机自启 | `app.setLoginItemSettings()` | 登录时自动启动 |
| 文件系统 | Node.js `fs` 模块 | 记忆/配置读写 |
| 自动更新 | `electron-updater` | 增量更新 |
