# 原生功能

## 桌面端原生能力

| 能力 | 实现方式 | 用途 |
|------|---------|------|
| 透明窗口 | Tauri 配置 `transparent: true` | 宠物窗口无背景 |
| 鼠标穿透 | `window.set_ignore_cursor_events()` | 点击穿透到底层应用 |
| 置顶 | Tauri 配置 `alwaysOnTop: true` | 宠物始终可见 |
| 窗口位置记忆 | 持久化 `window_position.json` | 关闭后恢复位置 |
| 多屏边界保护 | 窗口位置 clamped (x ≥ 0, y ≥ 0) | 窗口不超出屏幕边界 |
| 系统托盘 | `tauri::tray::TrayIconBuilder` | 菜单栏常驻图标 |
| Sidecar 通信 | stdin/stdout JSON | 与 Go PetCore 子进程通信 |
| 自动更新 | `tauri-plugin-updater` | 增量更新（Phase 2） |
