# 桌面宠物 UI 重新设计 — 恬静田园风格

> 日期: 2026-07-12
> 状态: 已定稿

## 1. 设计决策记录

### 1.1 设计目标
- 氛围: **可爱温馨**（暖色调、田园感、治愈系）
- 灵感来源: 星露谷物语 / 牧场物语
- 风格: 田园插画风格（非像素），保留透明窗口特性
- 范围: 全部 UI 重新设计（主界面、聊天气泡、设置面板、托盘菜单）

### 1.2 技术选型
| 决策 | 选择 | 理由 |
|------|------|------|
| CSS 框架 | **Tailwind CSS v4** + **daisyUI** | daisyUI 支持自定义暖色主题，可一键切换配色 |
| 动画库 | **Motion** (原 Framer Motion) + **AutoAnimate** | 声明式动画，React 原生支持，气泡/面板/列表动效 |
| 图标库 | **Lucide React** | 1600+ 图标，风格简洁统一 |
| 字体 | **Zen Maru Gothic** (Google Fonts) | 可爱圆体，贴合田园温馨感 |
| 组件策略 | 纯 Tailwind + daisyUI 预设组件，无需 shadcn/ui | 组件简单（气泡/按钮/面板），保持轻量 |

### 1.3 参考项目
- **BongoCat** (★ 21.8k) — Tauri 架构参考，透明窗口桌宠实现
- **openpets** (★ 889) — 桌面宠物平台架构、MCP 代理集成参考
- **clawd-on-desk** (★ 5.3k) — 自定义主题系统、状态驱动动画架构

## 2. 设计 Token

### 2.1 暖色田园色板

| Token | 值 | 用途 |
|-------|-----|------|
| `--color-bg` | `#FFF8E7` 奶油白 | 主背景色 |
| `--color-primary` | `#A8C5A0` 抹茶绿 | 主强调色 / 草地元素 |
| `--color-accent` | `#D4956A` 陶土橙 | 暖调强调色 / 按钮悬停 |
| `--color-secondary` | `#B5D8E8` 天空蓝 | 聊天气泡背景 |
| `--color-text` | `#4A3728` 深棕 | 文字色（代替纯黑） |
| `--color-bg-soft` | `#E8D5C4` 米色 | 次级背景 |
| `--color-green-light` | `#D4E0C8` 淡草绿 | 装饰元素 |
| `--color-peach` | `#F5D0B8` 蜜桃色 | 用户气泡背景 / 高亮 |

**配色原则：**
- ❌ 不使用纯黑 `#000000` — 用深棕 `#4A3728` 代替
- ❌ 不使用纯白 `#FFFFFF` — 用奶油白 `#FFF8E7` 代替
- ✅ 冷暖平衡：暖色（陶土橙、蜜桃）与冷色（天空蓝）搭配
- ✅ 自然色系：所有颜色从自然场景中提取
- ✅ 高对比度保证可读性

### 2.2 圆角体系

| Token | 值 | 用途 |
|-------|-----|------|
| `--radius-sm` | 8px | 按钮、输入框 |
| `--radius-md` | 16px | 聊天气泡、卡片 |
| `--radius-lg` | 24px | 面板、大容器 |

### 2.3 字体体系

| 层级 | 字体 | 用途 |
|------|------|------|
| 标题 | Georgia / Songti SC (衬线体) | 温暖手写感 |
| 正文 | PingFang SC / Microsoft YaHei (无衬线体) | 清晰可读 |
| 装饰元素 | Zen Maru Gothic (圆体) | 品牌文字、标签 |
| 等宽辅助 | monospace | 代码、时间戳 |

### 2.4 动画规范

| 元素 | 交互动效 | 时长 |
|------|---------|------|
| 按钮悬停 | 轻微放大 (scale 1.05) + 加深颜色 | 0.15s |
| 聊天气泡 | 淡入 + 向上弹入 (translateY 8px) | 0.3s |
| 面板弹出 | 缩放淡入 (scale 0.95 → 1) | 0.2s |
| 宠物表情切换 | 交叉淡入 + 轻微弹跳 | 0.25s |

## 4. 布局结构

```
┌─────────────────────────────────────┐
│  🌿 🌻 🌱   (装饰花草层)           │
│                                     │
│       🐱  (宠物区域，居中)          │
│     坐在草地上~                     │
│                                     │
│  ┌──────────────────┐               │
│  │ 早安！今天想做点  │ ← 聊天气泡   │
│  │ 什么呢？☀️       │   (天空蓝)    │
│  └──────────────────┘               │
│            ┌──────────────────┐     │
│            │ 帮我查一下天气~  │ ← 用户气泡
│            └──────────────────┘     │  (蜜桃色)    │
│                                     │
│  [💬 聊天]  [⚙️ 设置]  [🎮 互动]  │ ← 底部操作栏
└─────────────────────────────────────┘
```

**布局关键点：**
- 装饰层在顶部，强化田园氛围但不干扰操作
- 宠物在视觉中心，"坐"在草地色块上
- 宠物气泡在左（蓝色），用户气泡在右（蜜桃色）
- 底部操作栏为圆角药丸按钮
- 保留透明窗口特性

## 6. 迁移策略

1. **安装依赖** — Tailwind CSS + daisyUI + Motion + Lucide + AutoAnimate
2. **配置主题** — 创建 daisyUI `pastoral` 主题
3. **新建组件** — 用新风格重建核心 UI 组件
4. **渐进替换** — 逐个组件替换旧 CSS，不一次性重构
5. **清理旧 CSS** — 新组件覆盖后删除对应旧样式
