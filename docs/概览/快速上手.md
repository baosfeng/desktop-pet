# 快速上手

## 开发环境搭建

### 前置要求

- **Rust** 1.80+（含 `cargo`）
- **Go** 1.22+
- **Node.js** 20+（推荐 22+）
- **pnpm** 9+（`npm i -g pnpm`）
- **Tauri CLI**：`cargo install tauri-cli`

### 启动 PetCore（独立运行调试）

```bash
cd petcore
go run ./cmd/petcore/ --mode sidecar
```

### 启动前端 + Tauri（完整开发环境）

```bash
# 一步启动（自动构建前端 + 编译 Go + 运行 Rust 壳）
cd src-tauri
cargo tauri dev
```

### 分步调试

```bash
# 终端 1：Go PetCore
cd petcore
go run ./cmd/petcore/ --mode sidecar

# 终端 2：前端（Vite 开发服务器）
cd frontend
pnpm install
pnpm dev
```

### 打包 .app

```bash
# 1. 编译 Go PetCore
cd petcore
CGO_ENABLED=0 GOOS=darwin GOARCH=arm64 \
  go build -o ../src-tauri/binaries/petcore-aarch64-apple-darwin ./cmd/petcore/

# 2. 构建前端
cd frontend
pnpm build

# 3. 打包 .app（自动包含 petcore + 前端）
cd ../src-tauri
cargo tauri build

# 产物：src-tauri/target/release/bundle/dmg/DesktopPet-1.0.0.dmg (~25MB)
```

## 用户使用

1. 下载 `DesktopPet.dmg`（~25MB）并拖入 Applications
2. 双击打开 → 宠物出现在桌面
3. 右键宠物托盘图标 → 设置 → 填入 API Key 和模型名
4. 点击宠物 → 输入文字聊天
5. 右键宠物 → 「检查更新」→ 自动更新

## 目录结构

```
desktop-pet/
├── src-tauri/              # Tauri v2 Rust 壳
│   ├── src/                # Rust 源码（窗口/托盘/sidecar/命令）
│   ├── Cargo.toml
│   └── tauri.conf.json
├── frontend/               # React + PixiJS + Live2D 前端
│   ├── src/
│   │   ├── components/     # React 组件
│   │   ├── hooks/          # Tauri IPC hooks
│   │   └── lib/bridge.ts   # Tauri IPC 封装
│   └── package.json
├── petcore/                # Go PetCore 内核
│   ├── cmd/petcore/main.go # 入口
│   └── internal/           # 核心逻辑
│       ├── core/           # 状态引擎
│       ├── agent/          # AI Agent
│       ├── memory/         # 记忆系统
│       ├── llm/            # LLM 适配
│       ├── plugin/         # 插件系统
│       └── server/         # sidecar 通信
└── docs/                   # 项目文档
```
