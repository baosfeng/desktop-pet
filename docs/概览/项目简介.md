# 项目简介

## 一句话

**desktop-pet** 是一个像 QQ 宠物一样在桌面上陪伴你的 AI 伙伴。它接入 AI 大模型，越用越懂你，最终目标是能通过自然对话控制电脑。

## 核心特性

- **桌面陪伴**：Live2D 宠物常驻桌面，可拖拽/点击/互动
- **AI 对话**：接入 OpenAI 兼容 API，支持自然语言聊天
- **渐进记忆**：短期/核心/长期三层记忆系统，越用越懂你
- **隐私优先**：所有记忆数据存储在本地，不上传云端
- **工具扩展**：通过 MCP 协议扩展能力（联网搜索、文件操作等）
- **增量更新**：tauri-plugin-updater 自动增量更新，小白用户零门槛
- **极致轻量**：Tauri v2 + Go PetCore 架构，安装包仅 ~15-25MB，运行时内存仅 ~30-60MB

## 目标用户

想要一个聪明、可爱的桌面宠物陪伴自己的 Mac 用户。不需要懂编程，下载即用。
